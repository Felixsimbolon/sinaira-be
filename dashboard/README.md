# Dashboard Module — Sinaira Spa Management System

Nama: Jeremi Felix Adiyatma
NPM: 2306219575

## Overview

Dashboard merupakan modul frontend–analytics yang menyediakan ringkasan, visualisasi, dan insight strategis berdasarkan seluruh data operasional dalam sistem Sinaira.

Modul ini bertanggung jawab atas:

1. Penyajian KPI bisnis secara visual dan ringkas

2. Monitoring loyalitas dan performa membership customer

3. Penyediaan laporan operasional harian secara terpusat

4. Visualisasi tren booking, revenue, performa therapist, dan kondisi inventory

Dashboard ini terintegrasi langsung dengan seluruh modul backend berbasis Django, termasuk Booking Management, Membership & Loyalty, Therapist Performance, serta Inventory Management. Seluruh data ditampilkan dalam bentuk agregasi dan visualisasi tanpa mengubah data transaksi utama.

Tujuan utama modul ini adalah memastikan Owner dan Supervisor dapat memantau kondisi bisnis secara cepat, akurat, dan berbasis data tanpa perlu melakukan rekap manual.
